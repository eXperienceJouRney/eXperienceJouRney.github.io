整体工作流程

  1. 构建世界地图（使用SLAM）
  2. 红球视觉识别（OpenCV识别红色目标）
  3. 坐标转换（将像素坐标转换为世界坐标）
  4. 获取自身位置（从 /odometry/turtlebot 话题）
  5. 输出结果

一、构建世界地图（SLAM）

步骤1：启动TurtleBot

步骤2：运行SLAM（GMapping）

步骤3：在RViz中构建地图

  1. 启动RViz：

  1. RViz设置（添加以下显示项）：

    • Maps → 选择/map
    • TF → 选择/tf
    • Robot Model → 设置Fixed Frame: map，Robot Namespace: /turtletbot
    • Interactive Marker → 启用（用于控制机器人）

  2. 移动TurtleBot
    用键盘控制TurtleBot移动，让它覆盖整个环境。SLAM会实时生成地图。

  3. 保存地图（可选，但推荐）
    在RViz中点击 Save Map，保存为.pgm和.yaml文件（后续可加载）。

二、红球视觉识别与定位

原理

  1. 识别红球
    使用OpenCV在RGB图像中识别红色目标（HSV颜色空间）。
  2. 获取深度值
    通过深度相机（Kinect）获取红球在3D空间中的距离。
  3. 坐标转换
    • 将像素坐标 → 相机坐标系
    • 通过TF 转换到世界坐标系（map）

三、查看结果

1. 输出到终端

运行节点后，你会在终端看到：

  • Self Position：TurtleBot当前位置（世界坐标系）
  • Red ball detected：红球在图像中的像素位置
  • Red Ball Position：红球在世界坐标系中的位置

2. 在RViz中可视化

  1. 添加显示项：

    • Published Topic → 添加 /red_ball_pose（类型：PointStamped）
    • Published Topic → 添加 /self_pose（类型：PoseStamped）

  2. 设置框架

    • 所有话题的 Fixed Frame 设置为 map

  3. 效果

    • 红球位置会以红色点显示在地图上
    • TurtleBot位置会以绿色三角形显示
